This Privacy Policy explains how data is handled when you use the YouTube Content Analyzer GPT and its backend API.

## What data is sent to the backend
When a user enters a YouTube URL, the GPT sends only the following information to the backend API:
- The YouTube video URL (or video ID)

No other user messages or conversation context are transmitted to the backend.

## What data is NOT collected or stored
The backend API does NOT collect, store, share, or sell:
- User names
- Email addresses
- IP addresses
- Account details
- Chat history or conversations
- Video transcripts generated by the service
- Logs linking users to specific requests

All processing is performed in memory during the request.  
No data is saved to a database or persistent storage.

## Purpose of the data
The YouTube URL is used only to:
- Retrieve the chapter-wise transcript for the video
- Return it to the GPT so that the user can receive summaries and insights

No profiling, marketing, or analytics are performed.

## Third-party services
The backend fetches video transcripts from YouTube.  
Users are responsible for ensuring they have rights to access and process the video content.

## Security
Communication between the GPT and the backend occurs over HTTPS.

## Data retention
No data is retained after the request is complete.

## Children's privacy
This service is not intended for children under 13.
